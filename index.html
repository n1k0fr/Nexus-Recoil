FAQ

Common Questions.

Read this before asking in Discord — it saves everyone time.

Will this get me banned in Rainbow Six Siege?
Nexus Recoil uses standard Windows mouse input APIs that thousands of legitimate apps use daily. It does not inject into the game, read game memory, or use kernel drivers. That said, any third-party tool that modifies gameplay input is against Ubisoft's ToS and is entirely at your own risk. We make no guarantees. Use responsibly.
Why is it free? What's the catch?
No catch. This is a community project. The code is fully open source, there are no ads, no subscriptions, and no hidden data collection. We built it because the community deserves a safe, trustworthy free tool after seeing how many RATs are disguised as gaming utilities.
How do I know it's not a virus?
Read the full source code on GitHub before downloading. Every release has a SHA256 hash posted here — verify it on VirusTotal yourself. We also document exactly which Windows APIs are used: mouse_event and GetAsyncKeyState. Both are standard, non-suspicious, and used by thousands of legitimate programs.
The preset values don't feel right for my setup. What do I do?
Presets are starting points. Every DPI, sensitivity, and resolution combo is different. Load a preset as a base, tune the sliders until it feels right for you, then save it as a Custom preset so you never have to redo it.
Does it work with side mouse buttons as the aim key?
Yes. The aim key dropdown supports Right Mouse Button, Left Mouse Button, Middle Mouse Button, Side Button 1 (Mouse 4) and Side Button 2 (Mouse 5). Pick whichever you use to ADS in-game.
My antivirus flagged it. Is that normal?
Some antivirus tools flag programs using mouse input APIs as potentially suspicious — this is a false positive. Verify by reading the source code on GitHub. We submit the exe to major AV vendors for whitelisting with every release. If you are unsure, do not run it. Your safety matters more than using our tool.
Can I contribute or suggest new features?
Absolutely. Open an issue or pull request on GitHub, or post in our Discord. This is a community tool and contributions of all kinds are welcome.
